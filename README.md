# 파이썬 계산기

간단한 GUI 계산기 애플리케이션입니다. Python의 tkinter 라이브러리를 사용하여 만들어졌습니다.

## 기능

- 기본 산술 연산 (덧셈, 뺄셈, 곱셈, 나눗셈)
- 제곱근 계산
- 지우기 및 백스페이스 기능
- 사용자 친화적인 인터페이스

## 요구 사항

- Python 3.x
- tkinter 라이브러리 (대부분의 Python 설치에 기본 포함)

## 실행 방법

```bash
python calculator.py
```

## 사용 방법

1. 숫자 버튼(0-9)을 클릭하여 숫자를 입력합니다.
2. 연산자 버튼(+, -, *, /)을 클릭하여 연산을 선택합니다.
3. "=" 버튼을 클릭하여 결과를 계산합니다.
4. "C" 버튼을 클릭하여 모든 입력을 지웁니다.
5. "←" 버튼을 클릭하여 마지막 문자를 지웁니다.
6. "√" 버튼을 클릭하여 숫자의 제곱근을 계산합니다.
